使用C语言实现的功能丰富的PID控制器。

## Features

- [x] default （默认）
- [x] integral decay （积分衰减）
- [x] integral clamp （积分限幅）
- [x] integral separation （积分分离）
- [x] integral sliding window （积分滑动窗口，积分有限时间）
